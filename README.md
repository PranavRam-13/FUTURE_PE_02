FUTURE_PE_02 – AI UGC Ad Content System for Local Gym
Task Description

This project was created as part of the Future Interns Prompt Engineering Internship – Task 2.

The objective was to design a structured AI prompt system that generates high-conversion UGC-style short-form ads ready for Instagram Reels, Ads, and YouTube Shorts.

The system focuses on:

Scroll-stopping hooks

Relatable UGC scripts

Conversion-driven captions

Audience-specific messaging

Platform-ready formatting

Business Context

Business Type: Local Mid-Range Gym
Target Audience:

Beginners (20–35 years old)

Young professionals

People struggling with consistency

Individuals intimidated by gym environments

Core Audience Pain Points

Inconsistency with workouts

Feeling judged or intimidated

Lack of visible results

No structured guidance

Prompt Architecture

The system is modular and divided into three components:

1️ Hook Generator

Designed to create short, high-impact hooks under 12 words.
Focuses on:

Pattern interruption

Pain amplification

Curiosity triggers

Emotional relatability

Purpose: Capture attention within the first 3 seconds.

2️ UGC Script Generator

Structured using a proven short-form conversion format:

Hook

Problem

Personal struggle

Transformation

Specific result

Clear Call-to-Action

Purpose:
Build trust through relatability and transformation storytelling.

3️ Caption & CTA Generator

Generates:

Benefit-driven captions

Strong action prompts

Relevant hashtags

Purpose: Reinforce conversion and improve engagement.

Psychological Principles Applied

Pattern interruption (to stop scrolling)

Pain → Solution framing

Social proof via testimonial format

Transformation-based storytelling

Direct response CTA placement

AI Tools Used

ChatGPT – Hook, script, and caption generation

GitHub – Documentation and version control
